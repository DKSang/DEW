# Project Brief Checklist

- [ ] Project type selected.
- [ ] Target consumer identified.
- [ ] Business decision identified.
- [ ] Expected data product described.
- [ ] MVP scope defined.
- [ ] Non-goals defined.
- [ ] Learning objectives included.

- [ ] Evidence required before architecture listed.

- [ ] Learning log updated if learning mode is enabled.
