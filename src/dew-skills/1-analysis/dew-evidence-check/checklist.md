# Evidence Check Checklist

Use this checklist before marking a DEW phase complete.

## Required

- [ ] Target phase is identified.
- [ ] Target artifact is identified.
- [ ] Required evidence is listed.
- [ ] Existing evidence is inspected.
- [ ] Evidence is classified as missing/available/validated/stale/invalid.
- [ ] Validation evidence index is checked.
- [ ] Decision log is checked.
- [ ] Learning log is checked.
- [ ] Missing evidence triggers HALT-18.
- [ ] Caveated continuation requires user approval.
- [ ] Final evidence report is produced.

## Failure conditions

Fail the evidence check if:
- artifact exists but evidence does not
- evidence is not indexed
- evidence is stale or invalid
- phase is marked done with unresolved hard HALT
- caveated continuation was not explicitly approved by user
