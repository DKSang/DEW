# KPI Discovery Checklist

- [ ] Business discovery loaded or context provided.
- [ ] Business questions extracted.
- [ ] KPI candidates proposed.
- [ ] Each KPI has an ID.
- [ ] Each KPI maps to a business question.
- [ ] Each KPI has a formula.
- [ ] Each KPI has a grain.
- [ ] Each KPI lists required fields.
- [ ] Each KPI lists candidate source dependencies.
- [ ] Feasibility status is not marked verified.
- [ ] Assumptions are labeled.
- [ ] KPI-source matrix created.
- [ ] Decision log updated.
- [ ] Learning log updated if learning mode is enabled.
