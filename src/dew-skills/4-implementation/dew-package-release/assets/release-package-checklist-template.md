# Release Package Checklist Template

```markdown
# Release Package Checklist

## Required

- [ ] Skill QA passed or passed with caveats.
- [ ] module.yaml included.
- [ ] module-help.csv included.
- [ ] All required skill folders included.
- [ ] No critical QA findings remain.
- [ ] Version selected.
- [ ] Release notes created.
- [ ] Installation guide planned.
- [ ] Usage guide planned.
- [ ] Caveats documented.

## Optional

- [ ] Example project included.
- [ ] Demo walkthrough included.
- [ ] Screenshots included.
- [ ] Migration guide included.
```
