# Versioning Rubric

## 0.x pre-release

Use when:
- skill pack is useful but not stable
- workflow may still change
- user testing is expected

## 1.0 initial stable release

Use when:
- core workflow is complete
- install docs exist
- QA passed
- no critical/high blockers remain

## Patch

Use when:
- bug fixes
- typo/path fixes
- no workflow change

## Minor

Use when:
- new skill
- new template
- backward-compatible improvement

## Major

Use when:
- workflow order changes
- artifact structure changes
- breaking change to skill behavior
