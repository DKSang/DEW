# Package Release Risk Rubric

## Critical

Blocks package release.

Examples:
- module.yaml missing
- module-help.csv broken
- required skill missing
- install path unknown
- critical QA finding unresolved

## High

Usually blocks unless caveated.

Examples:
- broken optional flow
- unclear install guide
- missing key troubleshooting

## Medium

Can release with caveat.

Examples:
- example project missing
- minor documentation gaps

## Low

Does not block.

Examples:
- formatting/naming issue
