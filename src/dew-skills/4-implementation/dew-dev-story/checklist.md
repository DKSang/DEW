# Dev Story Checklist

- [ ] Story loaded.
- [ ] Story status is ready-for-dev.
- [ ] Story context pack loaded.
- [ ] Sprint status loaded.
- [ ] Dependencies checked.
- [ ] Implementation plan created.
- [ ] Tasks implemented.
- [ ] Required evidence produced or blocker documented.
- [ ] Validation commands/approach executed or blocker documented.
- [ ] DQ results produced if relevant.
- [ ] File list updated.
- [ ] Dev notes created.
- [ ] Evidence manifest created.
- [ ] Story updated.
- [ ] Sprint status updated to review or blocked.
