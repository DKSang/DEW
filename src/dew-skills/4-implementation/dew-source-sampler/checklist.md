# Source Sampler Checklist

- [ ] Source selected.
- [ ] Access evidence loaded or caveat recorded.
- [ ] Sample captured or missing reason documented.
- [ ] Sample notes created.
- [ ] Schema snapshot created.
- [ ] Required KPI fields mapped.
- [ ] Row/column summary documented.
- [ ] Time/entity/geography coverage documented where applicable.
- [ ] Caveats documented.
- [ ] Evidence index updated.
- [ ] HALT-18 triggered if required evidence is missing.
