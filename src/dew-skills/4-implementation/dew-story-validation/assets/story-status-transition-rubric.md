# Story Status Transition Rubric

## ready-for-dev → in-progress

Allowed by:
- dew-dev-story

## in-progress → review

Allowed when:
- implementation complete
- evidence produced or blocker documented

## review → done

Allowed only after:
- code review approved
- story validation passed
- Done Gate approved

## review → in-progress

Allowed when:
- review requests changes
- validation fails

## any → blocked

Allowed when:
- dependency missing
- evidence impossible
- design conflict exists
- user decision required
