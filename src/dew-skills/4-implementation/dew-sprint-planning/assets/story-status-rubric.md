# Story Status Rubric

## backlog

Story exists as candidate but does not yet have full story file.

## ready-for-dev

Story file exists and includes:
- context
- acceptance criteria
- implementation tasks
- evidence requirements
- validation commands
- DoD

## in-progress

Developer has started implementation.

## review

Implementation complete and awaiting review.

## done

Review passed, evidence exists, status updated.

## blocked

Story cannot proceed due to missing dependency, decision, or evidence.
