# Release Checklist Template

```markdown
# Release Checklist

## Required

- [ ] E2E validation passed or passed with caveats.
- [ ] Final evidence review complete or complete with caveats.
- [ ] No critical evidence gap remains.
- [ ] No critical DQ failure remains.
- [ ] Serving output exists if in scope.
- [ ] Trust and limitations statement exists.
- [ ] Governance/security review complete.
- [ ] DataOps runbook exists.
- [ ] Observability plan exists or caveated.
- [ ] Sprint status reviewed.
- [ ] Release scope defined.
- [ ] Release caveats defined.
- [ ] Release notes drafted.
- [ ] Rollback/recovery notes defined.
- [ ] User approved release decision.

## Optional

- [ ] Demo script exists.
- [ ] Screenshot/export exists.
- [ ] Version tag planned.
- [ ] README updated.
```
