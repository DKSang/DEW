# Sprint Status Update Rubric

## Story done update

A story can be marked done only when:
- story validation report says done or done-with-caveats
- evidence exists
- completion notes created

## Epic done update

An epic can be marked done only when:
- all required stories under epic are done
- no required story is blocked
- epic-level evidence exists or caveat approved

## Preserve status

Do not downgrade:
- done
- review
- in-progress

unless user explicitly requests correction.
