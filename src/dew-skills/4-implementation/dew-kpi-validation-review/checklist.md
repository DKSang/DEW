# KPI Validation Review Checklist

## Evidence reviewed

- [ ] KPI feasibility report reviewed.
- [ ] Field availability reviewed.
- [ ] Sample computation plan reviewed.
- [ ] Prototype reviewed.
- [ ] Sample output reviewed.
- [ ] Evidence index reviewed.

## Validation

- [ ] Required fields checked.
- [ ] Field meanings checked or caveated.
- [ ] Grain compatibility checked.
- [ ] Formula implementation checked.
- [ ] Output plausibility checked.
- [ ] Caveats documented.
- [ ] Findings classified by severity.

## Decision

- [ ] Recommended KPI status assigned.
- [ ] User approved final status.
- [ ] Decision log updated.
- [ ] Learning log updated if learning mode is enabled.
- [ ] Evidence index updated.
- [ ] Downstream impact documented.

## Failure conditions

Fail review if:
- evidence is missing
- formula is not implemented
- sample output does not exist
- required fields are missing and not caveated
- user has not approved final status
