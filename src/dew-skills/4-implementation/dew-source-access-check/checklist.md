# Source Access Check Checklist

- [ ] Source selected.
- [ ] Access method identified.
- [ ] Access target documented.
- [ ] Authentication/permission notes documented.
- [ ] Access attempt result recorded.
- [ ] Evidence path recorded.
- [ ] Failure/blocker documented if any.
- [ ] Evidence index updated.
- [ ] Source validation report update recommended.
- [ ] HALT-05 triggered if access remains unverified.
