# Step 05 — Review Decision Gate

## Mandatory Rules

- Stop at decision gate.
- Reviewer decision must be explicit.
- Critical finding blocks approval.

## Decision options

A. Approve  
B. Approve with caveats  
C. Request changes  
D. Block  

## Recommendation

Recommend one option.

## User/reviewer decision

Ask user to choose A/B/C/D.

Do not continue until user chooses.

## Status impact

- Approve / approve with caveats → story remains review, ready for validation
- Request changes → story returns to in-progress
- Block → story blocked

## Continue checkpoint

Ask:

`[C] Continue to finalize review`
