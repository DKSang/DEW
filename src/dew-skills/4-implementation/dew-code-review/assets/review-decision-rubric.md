# Review Decision Rubric

## Approve

Use when:
- ACs pass
- evidence exists
- no critical/high findings
- caveats documented

## Approve with caveats

Use when:
- no critical findings
- medium findings are acceptable
- caveats are documented
- validation still supports story goal

## Request changes

Use when:
- high findings exist
- missing non-critical evidence
- AC partially met
- correction is feasible

## Block

Use when:
- critical finding exists
- required evidence missing
- story contradicts design
- data output is unsafe/misleading
