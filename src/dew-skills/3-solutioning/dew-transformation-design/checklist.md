# Transformation Design Checklist

## Required inputs

- [ ] Storage design loaded.
- [ ] Ingestion design loaded.
- [ ] KPI feasibility report loaded.
- [ ] Source validation report loaded.
- [ ] Data product PRD loaded.
- [ ] Evidence index loaded or caveat recorded.

## Grain

- [ ] Input grains identified.
- [ ] Output grains identified.
- [ ] Grain transformations documented.
- [ ] User approved grain transformation decisions.
- [ ] Bridge/mapping needs identified.

## Bronze to Silver

- [ ] Bronze inputs listed.
- [ ] Silver outputs listed.
- [ ] Cleaning/casting/standardization rules defined.
- [ ] Metadata rules defined.
- [ ] Error/quarantine rules defined.
- [ ] DQ handoff defined.

## Silver to Gold

- [ ] Gold outputs listed.
- [ ] KPI mapping defined.
- [ ] Aggregation rules defined.
- [ ] Business rules defined.
- [ ] Caveats documented.

## Business rules

- [ ] Business rule catalog created.
- [ ] Each rule has ID.
- [ ] Each rule has owner or decision source.
- [ ] Each rule has change process.
- [ ] User approved critical business rules.

## Finalization

- [ ] transformation-design.md completed.
- [ ] business-rule-catalog.md completed.
- [ ] transformation-decision-record.md completed.
- [ ] Decision log updated.
- [ ] Learning log updated if enabled.
- [ ] No unresolved HALT-07 or HALT-14 remains.
