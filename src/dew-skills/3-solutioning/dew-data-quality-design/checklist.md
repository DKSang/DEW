- [ ] Source validation loaded.
- [ ] KPI feasibility loaded.
- [ ] Trust expectation loaded.
- [ ] DQ rules linked to source/model/KPI.

- [ ] Rule severity assigned.
- [ ] Rule action assigned.
- [ ] DQ gate policy created.
- [ ] DQ evidence requirements defined.
- [ ] User approved DQ rule gate.
- [ ] Decision log updated.
- [ ] Learning log updated if enabled.
