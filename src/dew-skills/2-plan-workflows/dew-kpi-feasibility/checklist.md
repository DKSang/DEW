# KPI Feasibility Checklist

## Required inputs

- [ ] Project brief loaded or context provided.
- [ ] Business discovery loaded or context provided.
- [ ] KPI catalog loaded.
- [ ] KPI-source matrix loaded.
- [ ] Data product PRD loaded or caveat recorded.
- [ ] Decision log loaded if available.
- [ ] Evidence index loaded if available.

## KPI definition checks

- [ ] KPI has ID.
- [ ] KPI has name.
- [ ] KPI maps to business question.
- [ ] KPI maps to consumer.
- [ ] KPI has formula.
- [ ] KPI has target grain.
- [ ] KPI lists required fields.
- [ ] KPI lists candidate source dependencies.

## Feasibility checks

- [ ] Required fields checked.
- [ ] Field meanings checked or marked unknown.
- [ ] Grain compatibility assessed.
- [ ] Sample computation plan created.
- [ ] Prototype requirement assigned.
- [ ] Feasibility status assigned.
- [ ] Caveats documented.
- [ ] User decision captured for primary KPIs.

## Evidence discipline

- [ ] KPI is not marked verified without sample computation.
- [ ] Missing evidence triggers HALT-03 or HALT-18.
- [ ] Evidence index updated if evidence exists.
- [ ] Decision log updated.
- [ ] Learning log updated if learning mode is enabled.
