# Source Discovery Checklist

- [ ] Project brief loaded or context provided.
- [ ] Data product PRD loaded or context provided.
- [ ] KPI catalog loaded or context provided.
- [ ] Candidate sources listed.
- [ ] Each source has source ID.
- [ ] Each source has source type.
- [ ] Each source has expected role.
- [ ] Each source maps to at least one KPI or product requirement.
- [ ] Each source has access method hypothesis.
- [ ] Each source has expected grain or marked unknown.
- [ ] Each source has expected freshness or marked unknown.
- [ ] Candidate status is not promoted to P1/P2.
- [ ] KPI-source coverage table created.
- [ ] Missing source for primary KPI triggers HALT-04.
- [ ] Assumptions are labeled.
- [ ] Decision log updated if decisions were made.
- [ ] Learning log updated if learning mode is enabled.
