# DEW Command Reference

## Quick Access

| Command / Skill | Purpose |
|---|---|
| `dew-project-brief` | Initialize a new project |
| `dew-requirement-gate` | Approve designs before architecture |
| `dew-create-data-story` | Prepare a task for implementation |
| `dew-dev-story` | Start coding an approved task |
| `dew-release-readiness` | Final release check |
| `bmad-help` | Check current project status and next steps |

## Metadata & Logs
| Skill | Output |
|---|---|
| `dew-decision-gate` | Logs major project decisions |
| `dew-learning-capture` | Updates `.learning-log.md` |
| `dew-evidence-check` | Verifies validation artifacts |
